# Complete Classical Sanskrit Training Course
## For English Speakers

**Course Duration:** 18-24 months for conversational proficiency  
**Time Commitment:** 1-2 hours daily recommended  
**Goal:** Speaking, reading, and writing Classical Sanskrit

---

## Course Structure Overview

### Phase 1: Foundation (Months 1-6)
- Devanagari script
- Basic grammar and sentence structure
- Essential vocabulary (500+ words)
- Simple reading and writing

### Phase 2: Intermediate (Months 7-12)
- Complex grammar and verb systems
- Expanded vocabulary (1500+ words)
- Reading simple texts
- Basic conversation

### Phase 3: Advanced (Months 13-18)
- Literary texts
- Advanced grammar and composition
- Fluent reading and writing
- Conversational proficiency

### Phase 4: Mastery (Months 19-24)
- Classical literature
- Poetry and drama
- Advanced composition
- Idiomatic expression

---

## PHASE 1: FOUNDATION (Months 1-6)

**Week 1-2: Vowels and Basic Consonants**

*Learning Objectives:*
- Master all 14 vowels (अ आ इ ई उ ऊ ऋ ॠ ऌ ॡ ए ऐ ओ औ)
- Learn first 10 consonants (क-त series)
- Understand vowel marks (मात्राs)

*Daily Practice (30 min):*
- Write each letter 20 times
- Practice pronunciation with audio resources
- Learn vowel combinations with consonants

*Exercises:*
1. Write your name in Devanagari
2. Copy simple words letter by letter
3. Recognize letters in isolation and in words

**Week 3-4: Remaining Consonants and Special Characters**

*Learning Objectives:*
- Complete all 33 consonants
- Learn conjunct consonants (संयुक्ताक्षर)
- Master anusvāra (ं), visarga (ः), and chandrabindu (ँ)

*Daily Practice (45 min):*
- Write complete consonant chart
- Practice conjunct consonants (क्त, त्र, ज्ञ, etc.)
- Read simple words aloud

*Exercises:*
1. Write 50 common conjunct consonants
2. Transcribe simple Sanskrit words you hear
3. Read basic two-syllable words

### Month 2: Basic Grammar Foundations

**Week 1: Nouns - Introduction to Cases**

*Concept: Sanskrit has 8 cases (vibhakti):*
1. Nominative (कर्ता) - subject
2. Accusative (कर्म) - object
3. Instrumental (करण) - by/with
4. Dative (सम्प्रदान) - to/for
5. Ablative (अपादान) - from
6. Genitive (सम्बन्ध) - of/possessive
7. Locative (अधिकरण) - in/on/at
8. Vocative (सम्बोधन) - O!/address

*Starting with Masculine A-stems (राम-type):*

Singular examples:
- रामः (Rāmaḥ) - Rama/nominative
- रामम् (Rāmam) - Rama/accusative
- रामेण (Rāmeṇa) - by Rama
- रामाय (Rāmāya) - to Rama
- रामात् (Rāmāt) - from Rama
- रामस्य (Rāmasya) - of Rama
- रामे (Rāme) - in Rama
- राम (Rāma) - O Rama!

*Daily Practice (60 min):*

**Week 3-4: Basic Sentence Construction**

*Word Order: Sanskrit is flexible but typical order is SOV (Subject-Object-Verb)*

Simple sentences:
1. रामः गच्छति। (Rāmaḥ gacchati.) - Rama goes.
2. बालः पुस्तकं पठति। (Bālaḥ pustakaṃ paṭhati.) - The boy reads a book.
3. नरः फलं खादति। (Naraḥ phalaṃ khādati.) - The man eats fruit.
4. गजः वने भवति। (Gajaḥ vane bhavati.) - The elephant is in the forest.

### Month 5: Complex Grammar Introduction

**Week 1-2: Compound Words (समास - Samāsa)**

*Types of compounds:*

1. **Dvandva (द्वन्द्व)** - Copulative
   - रामकृष्णौ (Rāma-Kṛṣṇau) = Rama and Krishna

2. **Tatpuruṣa (तत्पुरुष)** - Determinative
   - राजपुरुषः (rāja-puruṣaḥ) = king's man

3. **Karmadhāraya (कर्मधारय)** - Descriptive
   - नीलोत्पलम् (nīla-utpalam) = blue lotus

4. **Bahuvrīhi (बहुव्रीहि)** - Possessive
   - चतुर्भुजः (catur-bhujaḥ) = four-armed (one who has four arms)

*Daily Practice (60 min):*
- Identify compounds in texts
- Create your own compounds
- Break down complex compounds

**Week 3-4: Pronouns and Demonstratives**

*Essential Pronouns:*

1. **तद् (tad)** - he/she/it/that
   - सः (saḥ) - he
   - सा (sā) - she
   - तत् (tat) - it/that

2. **एतद् (etad)** - this
   - एषः (eṣaḥ) - this (m.)
   - एषा (eṣā) - this (f.)
   - एतत् (etat) - this (n.)

3. **अहम् (aham)** - I
   - मम (mama) - my
   - मया (mayā) - by me

4. **त्वम् (tvam)** - you (singular)
   - तव (tava) - your
   - त्वया (tvayā) - by you

*Daily Practice (60 min):*
- Decline pronouns fully
- Replace nouns with pronouns in sentences
- Write dialogues using pronouns

### Month 8: Verb System Deep Dive

**Week 1: The Ten Verb Classes (दश गणाः)**

*Understanding verbal roots and their classes:*

1. **भ्वादिः (bhvādi)** - भू-गण - thematic (-a-)
2. **अदादिः (adādi)** - अद्-गण - athematic
3. **जुहोत्यादिः (juhotyādi)** - हु-गण - reduplicating
4. **दिवादिः (divādi)** - दिव्-गण - thematic (-ya-)
5. **स्वादिः (svādi)** - सु-गण - -nu- class
6. **तुदादिः (tudādi)** - तुद्-गण - thematic (-a-)
7. **रुधादिः (rudhādi)** - रुध्-गण - -na/n- class
8. **तनादिः (tanādi)** - तन्-गण - -no/nu- class
9. **क्र्यादिः (kryādi)** - क्री-गण - -nā- class
10. **चुरादिः (curādi)** - चुर्-गण - causative (-aya-)

*Daily Practice (90 min):*
- Learn classification of 50 common verbs
- Conjugate verbs from different classes
- Recognize patterns in reading

**Week 2: Perfect Tense (लिट् लकार)**

*Formation varies by class but shows completed action*

Example with कृ (kṛ):
- चकार (cakāra) - he/she did
- चकर्थ (cakartha) - you did
- चकार (cakāra) - I did

*Daily Practice (60 min):*

**Week 3: Optative Mood (विधि लिङ्)**

*Expresses wish, potential, or mild command*

- भवेः (bhaveḥ) - may you be
- भवेत् (bhavet) - may he/she be

*Daily Practice (60 min):*
- Form optative of 20 verbs
- Express wishes and polite requests
- Read texts with optative usage

**Week 4: Imperative Mood (लोट् लकार)**

*Gives commands or requests*

Example with गम् (gam):
- गच्छानि (gacchāni) - let me go
- गच्छ (gaccha) - go! (you)

### Month 10: Sandhi Rules (संधि)

**Week 1: Vowel Sandhi**

*Rules for vowel combination at word boundaries*

Examples:
- अ + अ = आ: रामः अत्र → रामः अत्र → रामोत्र (becomes different form)
- Actual: देव + अत्र = देवात्र
- अ + इ = ए: तत्र + इति = तत्रेति
- अ + उ = ओ: राम + उवाच = रामोवाच

*Common Patterns:*
1. Similar vowels = long: इ + इ = ई
2. अ/आ + इ/ई = ए
3. अ/आ + उ/ऊ = ओ
4. अ/आ + ऋ = अर्
5. Hiatus removal with य्, व्, र्

*Daily Practice (60 min):*
- Apply sandhi rules to word pairs
- Break down sandhi in texts
- Practice both forming and splitting

**Week 2: Consonant Sandhi**

*Rules for consonant changes*

Key Rules:
1. Final -त् before voiced = -द्
2. Final -न् assimilates to following consonant
3. Visarga (ः) changes based on following sound
4. Doubling of consonants

*Daily Practice (60 min):*
- Practice 100 sandhi combinations
- Read unsandhied text and apply rules
- Break down sandhied text

**Week 3-4: Internal Sandhi**

*Sandhi within compound words*

*Daily Practice (60 min):*
- Analyze compounds with internal sandhi
- Form compounds correctly
- Practice recognition speed

### Month 11: Syntax and Advanced Grammar

**Week 1: Complex Sentences**

*Subordinate clauses using:*
- यदा...तदा (when...then)
- यदि...तर्हि (if...then)
- यत्र...तत्र (where...there)
- यः...सः (who/which...that)

Example:
```
यदा सूर्यः उदेति तदा पक्षिणः गायन्ति।
yadā sūryaḥ udeti tadā pakṣiṇaḥ gāyanti.

```

### Common Sandhi Rules

**Vowel Sandhi:**
1. a/ā + i/ī = e: रम + इति = रमेति
2. a/ā + u/ū = o: राम + उवाच = रामोवाच
3. a/ā + e = e: तव + एव = तवैव
4. a/ā + o = o: गच्छ + ओदनम् = गच्छोदनम्
5. Same vowels lengthen: देव + आगच्छति = देवागच्छति

**Visarga Sandhi:**
1. ः + voiced = o/r: नरः गच्छति = नरो गच्छति
2. ः + क/ख/प/फ/श/ष/स = remains or changes
3. Final -aḥ + a = o '

**Consonant Sandhi:**
1. -t + voiced = -d: तत् च = तद् च
2. -n assimilates: तान् च = तांश् च
3. -t/d + l = -l-l: उद् लोकः = उल्लोकः
